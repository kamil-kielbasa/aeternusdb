Expand description

Deserialize a value from a byte slice.

Returns (value, bytes_consumed) on success so that callers can advance a cursor through a buffer containing multiple encoded items.

Required Methods§

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
