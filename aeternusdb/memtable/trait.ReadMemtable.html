Expand description

Shared read interface for mutable and frozen memtables.

Allows engine code to treat an active Memtable and a FrozenMemtable uniformly for all read operations, without exposing the mutable write API.

§Object safety

The trait is object-safe — all methods take &self and return concrete types, so Box<dyn ReadMemtable> is valid.
