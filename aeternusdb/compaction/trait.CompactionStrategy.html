Expand description

A uniform interface for compaction strategies.

Each strategy receives the current set of SSTables (by reference), a mutable manifest for atomic metadata updates, the data directory, and the engine configuration. It returns:

  • Ok(Some(result)) — compaction was performed; the caller should update in-memory state using CompactionResult.
  • Ok(None) — nothing to compact (thresholds not met, etc.).
