Struct WalHeader

pub struct WalHeader {
    magic: [u8; 4],
    version: u32,
    max_record_size: u32,
    wal_seq: u64,
}
Expand description

Metadata written at the start of the WAL file.

This section validates the WAL’s identity and constraints. It is followed by a CRC32 checksum to protect against corruption.

Fields§

§magic: [u8; 4]

Magic constant to identify WAL files (b"AWAL").

§version: u32

WAL format version.

§max_record_size: u32

Maximum record size (in bytes).

§wal_seq: u64

Monotonically-increasing WAL sequence number (segment id).
