1//! # LSM Storage Engine
2//!
3//! This module implements a **synchronous**, **crash-safe** LSM-tree storage engine
4//! with multi-version concurrency, point and range tombstones, and pluggable
5//! compaction strategies.
6//!
7//! ## Design Overview
8//!
9//! The engine organises data across three layers, queried newest-first:
10//!
11//! 1. **Active memtable** — an in-memory sorted map backed by a write-ahead log (WAL).
12//! 2. **Frozen memtables** — read-only snapshots of previously active memtables,
13//!    awaiting flush to persistent SSTables.
14//! 3. **SSTables** — immutable, sorted, on-disk files with bloom filters and block
15//!    indices for efficient point lookups and range scans.
16//!
17//! Writes go through the WAL first, then into the active memtable. When the
18//! memtable exceeds [`EngineConfig::write_buffer_size`] it is frozen and a
19//! fresh memtable + WAL is created. Frozen memtables are flushed to SSTables
20//! via [`Engine::flush_oldest_frozen`] / [`Engine::flush_all_frozen`].
21//!
22//! ## Concurrency Model
23//!
24//! All engine state is protected by a single `Arc<RwLock<EngineInner>>`.
25//! Reads acquire a **read lock**; writes and flushes acquire a **write lock**.
26//! Compaction first acquires a short read lock to obtain the strategy, then
27//! acquires a write lock for the merge/swap phase.
28//!
29//! ## Compaction
30//!
31//! Three compaction operations are exposed:
32//!
33//! - [`Engine::minor_compact`] — merges similarly-sized SSTables within a
34//!   bucket, deduplicating point entries while preserving tombstones.
35//! - [`Engine::tombstone_compact`] — rewrites a single high-tombstone-ratio
36//!   SSTable, dropping provably-unnecessary tombstones.
37//! - [`Engine::major_compact`] — merges *all* SSTables into one, actively
38//!   applying range tombstones and dropping all spent tombstones.
39//!
40//! The concrete strategy implementations are selected via
41//! [`EngineConfig::compaction_strategy`].
42//!
43//! ## Guarantees
44//!
45//! - **Durability:** Every write is persisted to WAL before acknowledgement.
46//! - **Crash recovery:** On [`Engine::open`], the manifest, WALs, and SSTables
47//!   are replayed to reconstruct the last durable state.
48//! - **Multi-version reads:** Point lookups and scans always see the latest
49//!   committed version of each key, respecting tombstones.
50//! - **Atomic flushes:** Each frozen memtable is flushed to a single SSTable
51//!   and the manifest is updated atomically.

76/// Errors that can occur during engine operations.
77#[derive(Debug, Error)]
78pub enum EngineError {
79    /// Error originating from the manifest subsystem.
80    #[error("Manifest error: {0}")]
81    Manifest(#[from] ManifestError),
82
83    /// Error originating from the memtable subsystem.
84    #[error("Memtable error: {0}")]
85    Memtable(#[from] MemtableError),
86
87    /// Error originating from the SSTable subsystem.
88    #[error("SSTable error: {0}")]
89    SSTable(#[from] SSTableError),
90
91    /// Underlying filesystem I/O error.
92    #[error("I/O error: {0}")]
93    Io(#[from] std::io::Error),
94
95    /// Internal invariant violation (poisoned lock, unexpected state, etc.).
96    #[error("Internal error: {0}")]
97    Internal(String),
98}

100/// Configuration for an [`Engine`] instance.
101///
102/// Controls memtable sizing, compaction strategy selection, and all
103/// compaction-related thresholds. Passed to [`Engine::open`].
104pub struct EngineConfig {
105    /// Max memtable size (bytes) before freeze.
106    pub write_buffer_size: usize,
107
108    /// Compaction strategy to use for this engine instance.
109    ///
110    /// Determines which [`CompactionStrategy`](crate::compaction::CompactionStrategy)
111    /// implementations back the `minor_compact`, `tombstone_compact`, and
112    /// `major_compact` methods.
113    pub compaction_strategy: crate::compaction::CompactionStrategyType,
114
115    /// Lower bound multiplier for bucket size range ([avg × bucket_low, avg × bucket_high]).
116    pub bucket_low: f64,
117
118    /// Upper bound multiplier for bucket size range.
119    pub bucket_high: f64,
120
121    /// Min size (bytes) for regular buckets; smaller SSTables go to the "small" bucket.
122    pub min_sstable_size: usize,
123
124    /// Min SSTables in a bucket to trigger minor compaction.
125    pub min_threshold: usize,
126
127    /// Max SSTables to compact at once in minor compaction.
128    pub max_threshold: usize,
129
130    /// Ratio of tombstones to total records to trigger tombstone compaction.
131    pub tombstone_ratio_threshold: f64,
132
133    /// Min SSTable age (seconds) before eligible for tombstone compaction.
134    pub tombstone_compaction_interval: usize,
135
136    /// When true, tombstone compaction resolves bloom filter false positives
137    /// by doing an actual `get()` on other SSTables for point tombstones.
138    pub tombstone_bloom_fallback: bool,
139
140    /// When true, tombstone compaction will scan older SSTables to check
141    /// whether a range tombstone still covers any live keys, allowing
142    /// aggressive range tombstone removal.
143    pub tombstone_range_drop: bool,
144
145    /// Thread pool size for flushing memtables and compactions.
146    pub thread_pool_size: usize,
147}

242    /// Executes a memtable write operation, automatically freezing the active
243    /// memtable and retrying if the write buffer is full.
244    ///
245    /// Returns `Ok(true)` if a freeze occurred (caller should schedule a flush),
246    /// `Ok(false)` if the write succeeded without freezing.
247    fn write_with_retry(
248        inner: &mut EngineInner,
249        mut op: impl FnMut(&mut Memtable) -> Result<(), MemtableError>,
250    ) -> Result<bool, EngineError> {
251        match op(&mut inner.active) {
252            Ok(()) => Ok(false),
253            Err(MemtableError::FlushRequired) => {
254                Self::freeze_active(inner)?;
255                op(&mut inner.active)?;
256                let max_lsn = inner.active.max_lsn().unwrap_or(0);
257                inner.manifest.update_lsn(max_lsn)?;
258                Ok(true)
259            }
260            Err(e) => Err(e.into()),
261        }
262    }

268    /// Opens (or creates) an engine rooted at the given directory.
269    ///
270    /// On a fresh directory the manifest, WAL, and SSTable sub-directories
271    /// are created automatically. On an existing directory the manifest is
272    /// replayed, frozen WALs are loaded, and SSTables are opened.
273    pub fn open(path: impl AsRef<Path>, config: EngineConfig) -> Result<Self, EngineError> {
274        // 0. Create necessary directories
275        let base = path.as_ref();
276        let manifest_dir = base.join(MANIFEST_DIR);
277        let memtable_dir = base.join(MEMTABLE_DIR);
278        let sstable_dir = base.join(SSTABLE_DIR);
279
280        fs::create_dir_all(&manifest_dir)?;
281        fs::create_dir_all(&memtable_dir)?;
282        fs::create_dir_all(&sstable_dir)?;
283
284        // 1. Load or create manifest.
285        let manifest = Manifest::open(&manifest_dir)?;
286        let manifest_last_lsn = manifest.get_last_lsn()?;
287
288        // 2. Discover existing WAL files and load active/frozen WAL info from manifest.
289        let active_wal_nr = manifest.get_active_wal()?;
290        let active_wal_path = memtable_dir.join(format!("{:06}.log", active_wal_nr));
291        let memtable = Memtable::new(active_wal_path, None, config.write_buffer_size)?;
292
293        let frozen_wals = manifest.get_frozen_wals()?;
294        let mut frozen_memtables = Vec::new();
295        for wal_nr in frozen_wals {
296            let frozen_wal_path = memtable_dir.join(format!("{:06}.log", wal_nr));
297            let memtable = Memtable::new(frozen_wal_path, None, config.write_buffer_size)?;
298            frozen_memtables.push(memtable.frozen()?);
299        }
300
301        // 3. Discover existing SSTables on disk and remove orphans.
302        let sstables = manifest.get_sstables()?;
303
304        for entry in fs::read_dir(&sstable_dir)? {
305            let entry = entry?;
306            let file_path = entry.path();
307
308            if file_path.is_file()
309                && file_path.extension().and_then(|s| s.to_str()) == Some("sst")
310                && let Some(file_name) = file_path.file_name().and_then(|s| s.to_str())
311                && let Some(id) = file_name
312                    .strip_suffix(".sst")
313                    .and_then(|s| s.parse::<u64>().ok())
314                && !sstables.iter().any(|entry| entry.id == id)
315            {
316                fs::remove_file(&file_path)?;
317            }
318        }
319
320        // 4. Load SSTables from manifest.
321        let mut sstable_handles = Vec::new();
322        for sstable_entry in sstables {
323            let mut sstable = SSTable::open(&sstable_entry.path)?;
324            sstable.set_id(sstable_entry.id);
325            sstable_handles.push(sstable);
326        }
327
328        // 5. Compute max LSN across all sources.
329        let mut max_lsn = manifest_last_lsn;
330
331        if memtable.max_lsn().unwrap_or(0) > max_lsn {
332            max_lsn = memtable.max_lsn().unwrap_or(0);
333        }
334
335        for frozen in frozen_memtables.iter() {
336            if frozen.max_lsn().unwrap_or(0) > max_lsn {
337                max_lsn = frozen.max_lsn().unwrap_or(0);
338            }
339        }
340
341        for sstable in sstable_handles.iter() {
342            if sstable.max_lsn() > max_lsn {
343                max_lsn = sstable.max_lsn();
344            }
345        }
346
347        if memtable.max_lsn().unwrap_or(0) != max_lsn {
348            memtable.inject_max_lsn(max_lsn + 1);
349        }
350
351        // Sort frozen memtables by WAL sequence number, newest first.
352        // We use wal_seq rather than creation_timestamp because on crash
353        // recovery all frozen are replayed at nearly the same instant,
354        // making timestamps unreliable for ordering.
355        frozen_memtables.sort_by_key(|f| std::cmp::Reverse(f.wal_seq()));
356
357        // Sort SSTables by max_lsn descending.  This lets get()
358        // early-terminate: once we find a result at LSN L, any SSTable
359        // whose max_lsn ≤ L cannot contain a newer version of any key.
360        sstable_handles.sort_by_key(|s| std::cmp::Reverse(s.max_lsn()));
361
362        let inner = EngineInner {
363            manifest,
364            active: memtable,
365            frozen: frozen_memtables.into_iter().map(Arc::new).collect(),
366            sstables: sstable_handles.into_iter().map(Arc::new).collect(),
367            data_dir: base.to_path_buf(),
368            config,
369        };
370
371        Ok(Self {
372            inner: Arc::new(RwLock::new(inner)),
373        })
374    }

376    /// Gracefully shuts down the engine.
377    ///
378    /// Flushes all remaining frozen memtables, checkpoints the manifest,
379    /// and fsyncs all directories to ensure full durability.
380    pub fn close(&self) -> Result<(), EngineError> {
381        let mut inner = self.write_lock()?;
382
383        // 1. Flush any remaining frozen memtables to SSTables
384        while !inner.frozen.is_empty() {
385            Self::flush_frozen_to_sstable_inner(&mut inner)?;
386        }
387
388        // 2. Checkpoint the manifest to create a snapshot
389        let max_lsn = inner.active.max_lsn().unwrap_or(0);
390        inner.manifest.update_lsn(max_lsn)?;
391        inner.manifest.checkpoint()?;
392
393        // 3. Fsync directories to ensure metadata is durable
394        let manifest_dir = inner.data_dir.join(MANIFEST_DIR);
395        let memtable_dir = inner.data_dir.join(MEMTABLE_DIR);
396        let sstable_dir = inner.data_dir.join(SSTABLE_DIR);
397
398        // Fsync each directory
399        for dir_path in [&manifest_dir, &memtable_dir, &sstable_dir] {
400            if let Ok(dir) = fs::File::open(dir_path) {
401                dir.sync_all()?;
402            }
403        }
404
405        // 4. Fsync the root data directory
406        if let Ok(root) = fs::File::open(&inner.data_dir) {
407            root.sync_all()?;
408        }
409
410        Ok(())
411    }

413    /// Insert a key-value pair.
414    ///
415    /// Returns `Ok(true)` if the active memtable was frozen (caller should
416    /// arrange a flush), `Ok(false)` otherwise.

418        let mut inner = self.write_lock()?;

447    /// Look up a single key.
448    ///
449    /// Returns `Ok(Some(value))` if the key exists, `Ok(None)` if it has
450    /// been deleted or was never written, or `Err` on I/O failure.
451    ///
452    /// The lookup order is: active memtable → frozen memtables → SSTables
453    /// (all newest-first). The first definitive result wins.
454    pub fn get(&self, key: Vec<u8>) -> Result<Option<Vec<u8>>, EngineError> {
455        tracing::trace!(key_len = key.len(), "engine get");
456        let inner = self.read_lock()?;
457
458        // --------------------------------------------------
459        // 1. Active memtable (newest)
460        // --------------------------------------------------
461        match inner.active.get(&key)? {
462            MemtableGetResult::Put(value) => return Ok(Some(value)),
463            MemtableGetResult::Delete | MemtableGetResult::RangeDelete => return Ok(None),
464            MemtableGetResult::NotFound => {}
465        }
466
467        // --------------------------------------------------
468        // 2. Frozen memtables (newest → oldest)
469        // --------------------------------------------------
470        for frozen in &inner.frozen {
471            match frozen.get(&key)? {
472                MemtableGetResult::Put(value) => return Ok(Some(value)),
473                MemtableGetResult::Delete | MemtableGetResult::RangeDelete => {
474                    return Ok(None);
475                }
476                MemtableGetResult::NotFound => {}
477            }
478        }
479
480        // --------------------------------------------------
481        // 3. SSTables (sorted by max_lsn descending)
482        //
483        //    After size-tiered compaction, a merged SSTable may
484        //    span a wide LSN range. We track the best (highest-LSN)
485        //    result found so far. Once an SSTable's max_lsn is ≤
486        //    the best LSN, no subsequent SSTable can beat it, so
487        //    we break early.
488        // --------------------------------------------------
489        let mut best_sst: Option<sstable::GetResult> = None;
490        let mut best_lsn: u64 = 0;
491
492        for sst in &inner.sstables {
493            // Early termination: this SSTable (and all after it) have
494            // max_lsn ≤ best_lsn, so they can't contain a newer version.
495            if sst.max_lsn() <= best_lsn {
496                break;
497            }
498
499            match sst.get(&key)? {
500                sstable::GetResult::NotFound => {}
501                result => {
502                    let lsn = result.lsn();
503                    if lsn > best_lsn {
504                        best_lsn = lsn;
505                        best_sst = Some(result);
506                    }
507                }
508            }
509        }
510
511        match best_sst {
512            Some(sstable::GetResult::Put { value, .. }) => Ok(Some(value)),
513            Some(sstable::GetResult::Delete { .. } | sstable::GetResult::RangeDelete { .. }) => {
514                Ok(None)
515            }
516            _ => Ok(None),
517        }
518    }

520    /// Scan all live key-value pairs in `[start_key, end_key)`.
521    ///
522    /// Returns an iterator of `(key, value)` pairs, merging entries from
523    /// all layers and applying point/range tombstones to filter out
524    /// deleted keys.
525    pub fn scan(
526        &self,
527        start_key: &[u8],
528        end_key: &[u8],
529    ) -> Result<impl Iterator<Item = (Vec<u8>, Vec<u8>)>, EngineError> {
530        tracing::trace!(
531            start_len = start_key.len(),
532            end_len = end_key.len(),
533            "engine scan"
534        );
535        let merged = self.raw_scan(start_key, end_key)?;
536        Ok(VisibilityFilter::new(merged))
537    }

539    /// Captures an MVCC snapshot of all layers and merges them lazily.
540    ///
541    /// # MVCC snapshot approach
542    ///
543    /// 1. **Active memtable** — `.collect()` (mutable, already in RAM).
544    /// 2. **Frozen memtables / SSTables** — `Arc::clone` the handles
545    ///    (cheap pointer bump), then release the lock.
546    /// 3. Scan frozen memtables (in-RAM, scan already collects).
547    /// 4. Create **lazy** `ScanIterator<Arc<SSTable>>` per SSTable — reads
548    ///    blocks on demand via mmap, never materialising the full result
549    ///    set in RAM.
550    ///
551    /// The `Arc` keeps each layer alive even if a concurrent flush or
552    /// compaction removes it from `EngineInner` while we’re iterating.
553    fn raw_scan(
554        &self,
555        start_key: &[u8],
556        end_key: &[u8],
557    ) -> Result<utils::MergeIterator<'static>, EngineError> {
558        // --- snapshot under read lock (fast) ---
559        let (active_records, frozen_snapshot, sstable_snapshot) = {
560            let inner = self.read_lock()?;
561
562            // Active memtable — collect (mutable & in RAM, cheap).
563            let active_records: Vec<_> = inner.active.scan(start_key, end_key)?.collect();
564
565            // Clone Arc handles (pointer bumps, no data copy).
566            let frozen: Vec<Arc<FrozenMemtable>> = inner.frozen.iter().map(Arc::clone).collect();
567            let sstables: Vec<Arc<SSTable>> = inner.sstables.iter().map(Arc::clone).collect();
568
569            (active_records, frozen, sstables)
570        };
571        // --- lock released ---
572
573        let mut iters: Vec<Box<dyn Iterator<Item = Record>>> = Vec::new();
574
575        // Active memtable (already collected).
576        iters.push(Box::new(active_records.into_iter()));
577
578        // Frozen memtables — scan produces owned Records (in-RAM data).
579        for fm in &frozen_snapshot {
580            let records: Vec<_> = fm.scan(start_key, end_key)?.collect();
581            iters.push(Box::new(records.into_iter()));
582        }
583
584        // SSTables — lazy, block-at-a-time via mmap.
585        for sst in &sstable_snapshot {
586            let scan = SSTable::scan_owned(sst, start_key, end_key)?;
587            iters.push(Box::new(scan));
588        }
589
590        Ok(utils::MergeIterator::new(iters))
591    }

612    /// Freeze the current active memtable and swap in a fresh one.
613    /// The old memtable is pushed to the front of `inner.frozen`.
614    fn freeze_active(inner: &mut EngineInner) -> Result<(), EngineError> {
615        let frozen_wal_id = inner.active.wal_seq();
616        let current_max_lsn = inner.active.max_lsn().unwrap_or(0);
617        let new_active_wal_id = frozen_wal_id + 1;
618
619        let wal_path = inner
620            .data_dir
621            .join(MEMTABLE_DIR)
622            .join(format!("{:06}.log", new_active_wal_id));
623        let new_active = Memtable::new(wal_path, None, inner.config.write_buffer_size)?;
624
625        let old_active = std::mem::replace(&mut inner.active, new_active);
626        let frozen = old_active.frozen()?;
627        // Insert at beginning to maintain sorted order (newest first)
628        inner.frozen.insert(0, Arc::new(frozen));
629
630        // Ensure LSN continuity
631        inner.active.inject_max_lsn(current_max_lsn);
632
633        inner.manifest.add_frozen_wal(frozen_wal_id)?;
634        inner.manifest.set_active_wal(new_active_wal_id)?;
635
636        Ok(())
637    }

639    /// Flush the oldest frozen memtable to a new SSTable.
640    ///
641    /// Returns `Ok(true)` if a frozen memtable was flushed, `Ok(false)` if
642    /// there were no frozen memtables to flush.
643    pub fn flush_oldest_frozen(&self) -> Result<bool, EngineError> {
644        let mut inner = self.write_lock()?;
645
646        if inner.frozen.is_empty() {
647            return Ok(false);
648        }
649        Self::flush_frozen_to_sstable_inner(&mut inner)?;
650        Ok(true)
651    }

653    /// Flush **all** frozen memtables to SSTables.
654    ///
655    /// Returns the number of frozen memtables that were flushed.
656    #[allow(dead_code)]
657    pub fn flush_all_frozen(&self) -> Result<usize, EngineError> {
658        let mut inner = self.write_lock()?;
659
660        let mut count = 0usize;
661        while !inner.frozen.is_empty() {
662            Self::flush_frozen_to_sstable_inner(&mut inner)?;
663            count += 1;
664        }
665        Ok(count)
666    }

668    /// Allocates the next unique SSTable ID from the manifest's monotonic counter.
669    fn next_sstable_id(inner: &mut EngineInner) -> Result<u64, EngineError> {
670        Ok(inner.manifest.allocate_sst_id()?)
671    }
672
673    fn flush_frozen_to_sstable_inner(inner: &mut EngineInner) -> Result<(), EngineError> {
674        if inner.frozen.is_empty() {
675            return Ok(());
676        }
677
678        // Take the oldest frozen memtable (last in the newest-first vec).
679        // We flush oldest first so that `insert(0, sstable)` keeps the
680        // sstables list in newest-first order after a batch flush.
681        let frozen = inner
682            .frozen
683            .pop()
684            .ok_or_else(|| EngineError::Internal("frozen list became empty unexpectedly".into()))?;
685        let frozen_wal_id = frozen.wal_seq();
686
687        // Get all records from the frozen memtable and split into
688        // point entries and range tombstones via Record::into_entry().
689        let mut point_entries = Vec::new();
690        let mut range_tombstones = Vec::new();
691
692        for record in frozen.iter_for_flush()? {
693            match record.into_entry() {
694                RecordEntry::Point(pe) => point_entries.push(pe),
695                RecordEntry::Range(rt) => range_tombstones.push(rt),
696            }
697        }
698
699        // Generate unique SSTable ID and path
700        let sstable_id = Self::next_sstable_id(inner)?;
701        let sstable_path = inner
702            .data_dir
703            .join(SSTABLE_DIR)
704            .join(format!("{:06}.sst", sstable_id));
705
706        // Build the SSTable
707        let point_count = point_entries.len();
708        let range_count = range_tombstones.len();
709
710        sstable::SstWriter::new(&sstable_path).build(
711            point_entries.into_iter(),
712            point_count,
713            range_tombstones.into_iter(),
714            range_count,
715        )?;
716
717        // Load the newly created SSTable
718        let mut sstable = SSTable::open(&sstable_path)?;
719        sstable.set_id(sstable_id);
720        // Insert at beginning to maintain sorted order (newest first)
721        inner.sstables.insert(0, Arc::new(sstable));
722
723        // Update manifest
724        inner.manifest.add_sstable(ManifestSstEntry {
725            id: sstable_id,
726            path: sstable_path,
727        })?;
728
729        // Remove the frozen WAL from manifest
730        inner.manifest.remove_frozen_wal(frozen_wal_id)?;
731
732        Ok(())
733    }

739    /// Execute a compaction strategy, applying the result to the engine.
740    ///
741    /// Returns `Ok(true)` if compaction was performed, `Ok(false)` if
742    /// the strategy decided there was nothing to do.
743    fn run_compaction(
744        &self,
745        strategy: &dyn crate::compaction::CompactionStrategy,
746    ) -> Result<bool, EngineError> {
747        let mut inner = self.write_lock()?;
748
749        let inner = &mut *inner; // reborrow to split fields
750        let sst_count = inner.sstables.len();
751        let data_dir_str = inner.data_dir.to_string_lossy();
752        let result = strategy
753            .compact(
754                &inner.sstables,
755                &mut inner.manifest,
756                &data_dir_str,
757                &inner.config,
758            )
759            .map_err(|e| EngineError::Internal(format!("Compaction failed: {e}")))?;
760
761        match result {
762            None => {
763                tracing::debug!(sst_count, "compaction strategy found nothing to do");
764                Ok(false)
765            }
766            Some(cr) => {
767                tracing::info!(
768                    sst_count_before = sst_count,
769                    removed = cr.removed_ids.len(),
770                    new_id = ?cr.new_sst_id,
771                    "compaction applied"
772                );
773                Self::apply_compaction_result(inner, cr)?;
774                Ok(true)
775            }
776        }
777    }

779    /// Acquires the compaction strategy from the configuration and runs it.
780    ///
781    /// The `selector` function picks which strategy variant (minor, tombstone,
782    /// or major) to obtain from the configured [`crate::compaction::CompactionStrategyType`].
783    fn compact_with(
784        &self,
785        selector: fn(
786            &crate::compaction::CompactionStrategyType,
787        ) -> Box<dyn crate::compaction::CompactionStrategy>,
788    ) -> Result<bool, EngineError> {
789        let strategy = {
790            let inner = self.read_lock()?;
791            selector(&inner.config.compaction_strategy)
792        };
793        self.run_compaction(strategy.as_ref())
794    }

796    /// Runs one round of **minor compaction** (size-tiered).
797    ///
798    /// Selects the best bucket whose size exceeds `min_threshold` and merges
799    /// those SSTables into a single new SSTable, deduplicating point entries
800    /// and preserving all tombstones.
801    ///
802    /// Returns `Ok(true)` if compaction was performed, `Ok(false)` if no
803    /// bucket met the threshold.
804    pub fn minor_compact(&self) -> Result<bool, EngineError> {
805        self.compact_with(crate::compaction::CompactionStrategyType::minor)
806    }
807
808    /// Runs one round of **tombstone compaction** (per-SSTable GC).
809    ///
810    /// Selects the SSTable with the highest tombstone ratio that exceeds
811    /// `tombstone_ratio_threshold` and rewrites it, dropping provably-unnecessary
812    /// tombstones.
813    ///
814    /// Returns `Ok(true)` if compaction was performed, `Ok(false)` if no
815    /// SSTable was eligible.
816    pub fn tombstone_compact(&self) -> Result<bool, EngineError> {
817        self.compact_with(crate::compaction::CompactionStrategyType::tombstone)
818    }
819
820    /// Runs **major compaction** — merges all SSTables into one.
821    ///
822    /// Actively applies range tombstones to suppress covered Puts, and
823    /// drops all spent tombstones from the output.
824    ///
825    /// Returns `Ok(true)` if compaction was performed, `Ok(false)` if
826    /// there are fewer than 2 SSTables.
827    pub fn major_compact(&self) -> Result<bool, EngineError> {
828        self.compact_with(crate::compaction::CompactionStrategyType::major)
829    }

831    /// Applies a `CompactionResult` to the in-memory engine state.
832    ///
833    /// Removes consumed SSTables, inserts the newly built one, and
834    /// re-sorts by `max_lsn` descending so that `get()` can
835    /// early-terminate correctly.
836    fn apply_compaction_result(
837        inner: &mut EngineInner,
838        cr: crate::compaction::CompactionResult,
839    ) -> Result<(), EngineError> {
840        // Remove consumed SSTables.
841        inner
842            .sstables
843            .retain(|sst| !cr.removed_ids.contains(&sst.id()));
844
845        // Load and insert new SSTable if one was produced.
846        if let Some(ref path) = cr.new_sst_path {
847            let mut new_sst = SSTable::open(path)?;
848            new_sst.set_id(cr.new_sst_id.unwrap_or(0));
849            inner.sstables.push(Arc::new(new_sst));
850        }
851
852        // Re-sort by max_lsn descending to maintain the early-termination
853        // invariant used by get().
854        inner
855            .sstables
856            .sort_by_key(|s| std::cmp::Reverse(s.max_lsn()));
857
858        Ok(())
859    }
